# Happy Buddha Restaurant Website

Welcome to the Happy Buddha Restaurant website repository. This project is a web application for a renowned dim sum restaurant established 30 years ago in Vienna. The website provides information about the restaurant, including its menu, gallery, contact details, and more.

## About

Happy Buddha is a renowned dim sum restaurant established 30 years ago in the heart of Vienna. Our website offers visitors an overview of our menu, a gallery of our dishes, contact information, and the history of our restaurant.

## Features

- **Home Page**: Introduction to the restaurant and its offerings.
- **Menu Page**: Detailed menu with categories and images of dishes.
- **Gallery Page**: A gallery showcasing images of our dishes and restaurant.
- **About Us Page**: Information about the history and background of the restaurant.

## Technologies

- **Next.js**: React framework for server-side rendering and static site generation.
- **React**: JavaScript library for building user interfaces.
- **CSS Modules**: Scoped CSS for styling components.
- **TypeScript**: Typed superset of JavaScript for type safety.
- **Google Maps Embed API**: For embedding the restaurant's location on the contact page.
